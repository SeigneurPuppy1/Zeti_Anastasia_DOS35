Глоссарий для этой папки
1. Вывод скрипта output.sh находится в script_output.txt
2. В файле commands.txt находится подтверждение наличие команд из задания: htop, mc, tree
3. Вывод команды, который выводит список установленных пакетов dpkg --get-selections находится в installed_packages.txt
4. В файле службы sysinfo.service пришлось немного поменять структуру:
  - Удалить пункт Restart=always, так как происходит бесконечный цикл
  - Пришлось добавить строку для корректной записи логов StandardOutput=append:/var/log/sysinfo.log, так как метод описанный в примере не работает
5. Вывод записи лога на скриншоте "OutputLogService"
